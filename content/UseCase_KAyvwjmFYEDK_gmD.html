Cliente
Steps
1. El cliente accede a la pagina web
2. Inicia sesión con sus credenciales
3. Ingresa al modulo "Agendar cita"
4. El sistema consulta, muestra disponibilidad de agenda y muestra los servicios disponibles
5. El cliente selecciona el servicio y el profesional
6. if no hay disponibilidad o la fecha es inválida
    6.1. el sistema muestra mensaje de error
   end if
7. El cliente selecciona fecha y hora
8. El sistema pide confirmación
9. El cliente confirma la acción
10. El sistema pregunta si desea activar el recordatorio
11. if el cliente activa el recordatorio
    11.1. el sistema muestra las opciones (SMS o correo)
    11.2. el cliente confirma la selección
    11.3. el sistema muestra mensaje de éxito
    end if
12. El sistema registra la cita
13. El sistema muestra mensaje de confirmación

Administrador
Steps
1. El administrador accede a la pagina web
2. Inicia sesión con sus credenciales
3. Ingresa al modulo "Agendar cita"
4. El sistema consulta, muestra disponibilidad de agenda y muestra los servicios disponibles
5. El administrador selecciona el usuario que solicita la cita
6. El administrador selecciona el servicio y el profesional
7. El administrador selecciona fecha y hora
8. if no hay disponibilidad o la fecha es inválida
    8.1. el sistema muestra mensaje de error
   end if
9. El sistema pide confirmación
10. El usuario confirma la acción
11. El sistema pregunta si desea activar el recordatorio
12. if el usuario activa el recordatorio
    12.1. el sistema muestra las opciones (SMS o correo)
    12.2. el usuario confirma la selección
    12.3. el sistema muestra mensaje de éxito
    end if
13. El sistema registra la cita
14. El sistema muestra mensaje de confirmación
