Scenario
Steps
1. El administrador ingresa el URL o ingresa por medio de LINK de Naiapp
2. El administrador inicia sesión con sus credenciales en la sección "Soy parte de NaiApp"
3. Administrador accede a “Reportes”
4. El sistema muestra parámetros para generar el reporte de ingresos
5. El administrador modifica los parámetros (fecha inicio, fecha fin, profesional, servicio especifico)
6. El sistema valida nueva configuración
7. El sistema genera nuevamente el reporte
8. El sistema muestra versión actualizada
