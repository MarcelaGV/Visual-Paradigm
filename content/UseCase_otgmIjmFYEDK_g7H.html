Use Case Note
■ Workflow
    • // Write down briefly how user perform the work
    • El usuario selecciona el modulo "Calificaciones"
    • El sistema consulta las calificaciones y calcula el promedio
    • El sistema muestra los resultados
■ Business Logic
    • // Write down what user expect the system to react upon certain condition (e.g. low inventory alert level)
    • El sistema debe mostrar el promedio de calificaciones de los profesionales, los servicios y el establecimiento
    • El sistema debe mostrar la cantidad de reseñas
    • El promedio se calcula de acuerdo al numero de estrellas en las preguntas correspondientes
■ Decisions
    • // Write down the decisions made during the meeting (e.g. Must allow accessing from mobile devices)
    • Se decidió permitir filtro por fecha
    • Se decidió mostrar el promedio con un decimal
■ Follow-up
    • // Write down the items that should follow-up in the coming meeting
    • Definir si se va a mostrar calificación individual o general de los profesionales

Administrador y Profesional
Steps
1. El usuario accede a la pagina web
2. Ingresa al modulo "Calificaciones"
3. El sistema consulta las calificaciones y calcula el promedio
4. El sistema muestra el promedio de calificaciones de los profesionales, los servicios y el establecimiento
5. if si el usuario desea filtrar por fecha debe seleccionar "Filtrar"
    5.1. selecciona la fecha o rango
    5.2. el sistema filtra los resultados
   end if

Cliente
Steps
1. El cliente accede a la pagina web
2. if el cliente no quiere iniciar sesión
    2.1. el cliente hace scroll en el dashboard para llegar a la sección "Nuestros Profesionales" y "Lo Que Dicen Nuestros Clientes"
    2.2. el sistema muestra el promedio de calificaciones de profesionales y las calificaciones de otros clientes
   end if
3. Ingresa al modulo "Agendar cita"
4. Baja a la sección de "Calificaciones de Nuestros Profesionales"
5. El sistema consulta las calificaciones y muestra el promedio
6. El sistema muestra el promedio de calificaciones de los profesionales
