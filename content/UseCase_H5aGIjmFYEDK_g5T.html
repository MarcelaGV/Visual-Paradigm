Steps
1. El administrador accede a la pagina web
2. El administrador inicia sesión con sus credenciales en la sección "Soy parte de NaiApp"
3. Ingresa el modulo "Pagos"
4. Selecciona la opción "Nuevo pago"
5. El sistema muestra el formulario
6. El administrador ingresa los datos
7. if los datos estan incompletos
    7.1. el sistema no deja guardar pago
   end if
8. El sistema valida información
9. El sistema registra el pago
